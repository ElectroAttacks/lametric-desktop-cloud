# Privacy Policy

This application connects to the LaMetric Cloud API to access devices owned by the authenticated user.

## Data that is accessed
The application may access the following data:
- Device ID
- Device name
- Serial number
- Local IP address
- Device API key

## Purpose
This data is used only to:
- Discover LaMetric devices
- Allow the user to interact with their own devices

## Data storage
No personal data is stored permanently.
All data is processed in memory and discarded when the session ends.

## Data sharing
No data is shared with third parties.

## Revoking access
You can revoke access at any time from the LaMetric account settings by removing the account in the application settings.
